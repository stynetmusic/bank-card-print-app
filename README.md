# UF Print - Система для печати банковских карт

Приложение для создания коммерческих предложений на печать банковских карт размером 87x56мм.

## Возможности

- **Загрузка изображений**: Поддержка двух сторон карты (А и Б)
- **Редактирование**: Растяжение, уменьшение, ластик, коррекция цвета CMYK
- **Предварительный просмотр**: Мгновенный просмотр в реальном размере
- **Данные заказчика**: Ввод информации о клиенте и компании
- **Характеристики печати**: Выбор типа печати, материала, ламинации
- **История заказов**: Автоматическое сохранение всех заказов в базу данных
- **Экспорт PDF**: Сохранение в оригинальном качестве для печати

## Системные требования

- Windows 10/11 (x64)
- 4 GB RAM минимум
- 100 MB свободного места на диске

## Установка на Windows

### Вариант 1: Использование готового .exe файла (рекомендуется)

1. Скачайте файл `UF_Print_Cards.exe`
2. Запустите файл (возможно потребуется подтверждение SmartScreen)
3. Приложение готово к работе

### Вариант 2: Компиляция из исходного кода

#### Шаг 1: Установка Python

1. Скачайте Python с https://www.python.org/downloads/
2. Выберите версию Python 3.11 или 3.12
3. При установке **обязательно** поставьте галочку "Add Python to PATH"
4. После установки перезагрузите командную строку

#### Шаг 2: Установка зависимостей

Откройте командную строку (cmd) или PowerShell и выполните:

```bash
pip install PyQt6 Pillow reportlab pyinstaller
```

#### Шаг 3: Компиляция в .exe

1. Скопируйте все файлы проекта в одну папку
2. Откройте командную строку в этой папке
3. Выполните команду:

```bash
pyinstaller --onefile --windowed --name UF_Print_Cards main.py
```

Или используйте файл спецификации:

```bash
pyinstaller build.spec
```

4. Готовый .exe файл будет в папке `dist`

#### Шаг 4: Запуск

Запустите `UF_Print_Cards.exe` из папки `dist`

## Использование

### 1. Редактор карт

- Нажмите "Загрузить Сторона А" для загрузки лицевой стороны
- Нажмите "Загрузить Сторона Б" для загрузки обратной стороны
- Используйте инструменты для редактирования:
  - **Просмотр**: Обычный режим
  - **Растянуть**: Увеличение изображения
  - **Уменьшить**: Уменьшение изображения
  - **Ластик**: Стирание частей изображения (регулируется ползунком)
- **CMYK**: Настройка цветового баланса

### 2. Данные заказа

Перейдите на вкладку "Данные заказа" и заполните:
- Данные заказчика (имя, телефон, email)
- Данные компании
- Характеристики печати (тираж, тип печати, материал, ламинация)

### 3. Сохранение и экспорт

- Нажмите "Сохранить заказ" для сохранения в базу данных
- Нажмите "Экспорт в PDF" для создания файла для печати

### 4. История заказов

На вкладке "История заказов" можно:
- Просмотреть все сохраненные заказы
- Открыть любой заказ для редактирования
- Обновить список заказов

## Формат файлов

- **Входные**: PNG, JPG, JPEG, BMP, TIFF
- **Выходные**: PDF (оригинальное качество)

## Размер карт

Стандартный размер банковской карты: 87×56 мм

## База данных

Заказы сохраняются в файле `card_printing.db` (SQLite) в папке с программой.

## Устранение неполадок

### Ошибка при запуске .exe

- Убедитесь, что у вас установлены последние обновления Windows
- Отключите антивирус временно и попробуйте снова
- Запустите от имени администратора

### Ошибка при загрузке изображений

- Проверьте формат файла (должен быть PNG, JPG, JPEG, BMP или TIFF)
- Убедитесь, что файл не поврежден

### Ошибка при экспорте PDF

- Убедитесь, что есть права на запись в выбранную папку
- Проверьте свободное место на диске
